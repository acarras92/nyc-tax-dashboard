# NYC Property Tax Tracker

Live dashboard for tracking NYC hotel and commercial property tax assessments. Built for Highgate Capital deal diligence and portfolio monitoring.

**Live URL:** https://acarras92.github.io/nyc-tax-dashboard/

## Data Sources
- NYC Department of Finance Property Tax System
- ACRIS (Automated City Register Information System)

## Files
- `index.html` — Single-file dashboard (HTML + CSS + JS inline)
- `properties.json` — Property tax data (updated by scraper)

## Adding Properties
Add new property objects to `properties.json` following the existing schema, then push to deploy.
